Description
-----------

These problems are taken from the project euler website, https://projecteuler.net/about
Basically it is a series of math problems that are solved using programming.
For more information read the about page on their site.

The source code for each problem contains comments and descriptions of the problems. 

Execution
---------

So far all the problems have been solved using python, using version 3.4.
But they just need to be ran and then they will spit out the answer.
